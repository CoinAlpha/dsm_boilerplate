## Boiletplate project for CoinAlpha DSM

Bootstrapped with https://github.com/ben-rogerson/twin.examples/tree/master/react-emotion

Work in progress!


## Getting started

Checkout the project and run:

```shell
npm install
```
or
```shell
yarn
```

Then update it

```shell
yarn update
```

And then run it

```shell
yarn start
```

It will automatically open a new tab on
http://localhost:1234
